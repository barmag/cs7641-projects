Requirements for running the experiments
- Python 3.6
- Jupyter
- scikit-learn
- Pandas

Running the experiments
- Experiments are saved in a Jupyter notebook per algorithm
- Decision trees: decisiontree-experiments.ipynb
- Neural Networks: mlp_nn_experiments.ipynb
- Boosting: boosting_experiments.ipynb
- SVM: svm-experiments.ipynb
- KNN: knn-experiments.ipynb

The data is included under the data folder of the prject github repo https://github.com/barmag/cs7641-projects
